
# My Contacts

Contacts API created with nodeJS in JStack course


## Installation


To install you need to have node installed, with the project manager npm or yarn.

Project Clone:

```bash
  git clone https://github.com/vinicdev/mycontacts/
```

Install dependencies:
```bash 
  npm install
```

Run project:

```bash 
    npm run dev 
    
    or

    yarn dev
```
